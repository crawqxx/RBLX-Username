# RBLX-Username

Welcome to the **RBLX-Username** Python application! This tool is designed to help you generate and check "rare" Roblox usernames, including those with specific patterns like 5-symbols or 6-letters.

## Features

- **Fast Username Generation**: Generate new usernames in seconds!
- **Username Checker**: Check the status of usernames.
- **Account Creator (Windows only)**: Create Roblox accounts quickly using the app.
- **Generate Filters**: Customize username generation with filters, such as 5 symbols with 3 consecutive identical characters.
- **Open Source**: This project is fully open source. You can review and modify the code **with my permission**.

## Lite Version

If you don't need the Username Checker or Account Creator functionalities, consider using the Lite version.

### Lite Version Features:

- **Faster Speed**: Optimized for performance.
- **Reduced File Size**: Smaller download size.

## Requirements & Dependencies

To run the program, ensure you have [Python](https://www.python.org/downloads/) installed.

### Required Python Packages

Install the following packages using pip:

```bash
pip install selenium aiohttp colorama
```

**Note**: If you are using the Lite version, you do not need to install `selenium`.
